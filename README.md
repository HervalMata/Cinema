# Gerenciador de reservas de cinema

## Fazer reserva:

* Selecionar filme
* Selecionar Sessão
* Selecionar Assento
* Gerar bilhete de reserva

## Ver reserva:

* Buscar reserva por código do bilhete

# tecnologias utilizadas:

## JavaEE 7

* JSF 2.2
* PRIMEFACES 6.1
* OMNIFACES 2.6
* HIBERNATE 5
* JPA 2.1
* MAVEN 3.3
* Servidor de aplicação: WILDFLY 10.1.0
* Banco de dados: MYSQL( Connector/J 5.1.42 )
